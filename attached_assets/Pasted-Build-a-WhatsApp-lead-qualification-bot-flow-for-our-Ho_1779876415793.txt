Build a WhatsApp lead qualification bot flow for our Hotel CRM SaaS using Laravel + Livewire.

IMPORTANT:

* Human-like conversational flow
* Bot should always resume where conversation stopped
* Save every step in database
* Update contact details dynamically
* Create CRM lead card automatically
* Use warm natural language
* No robotic replies
* Must support STOP opt-out system

STOP / OPT OUT SYSTEM
=====================

If user types:
STOP
UNSUBSCRIBE
NO

Immediately:

* stop all campaigns
* set opt_out = true
* set lead_status = opted_out

Bot reply:
"You have been unsubscribed successfully 😊

If you need hotel CRM demo in future, simply message HI."

IMPORTANT:
Never send any message again after STOP.

Only restart flow if user messages again manually.
